# camera_bot — полный справочник команд (v2.9-J, 2026-07-09)

Telegram-бот **@your_bot** для эксплуатации парка ~634 IP-камер МФК «Зарядье».
**164 команды**, 123 алиаса (включая русские: `/диаг`, `/снимок`, `/поиск`…), 50 inline-кнопок, 44 модуля.
Владелец один (TOFU). Пароли камер бот не меняет. Все операции записи — через двухшаговое подтверждение.

Меню «/» в Telegram показывает 100 главных команд; полный список — здесь. `/help` в боте — с интерактивными табами по разделам.

## 📷 База: камеры, снимки, диагностика

| Команда | Что делает |
|---|---|
| `/start` | Онбординг-тур; deep-link `/start shot_10-20-50-1` сразу делает действие; из QR-наклейки открывает карточку камеры |
| `/help` | Справка с inline-табами по разделам |
| `/shot <ip\|имя>` | Снимок JPEG с камеры. Кнопки под фото: Повторить/Диаг/Инфо/📎 Оригинал (без пересжатия). Подпись — имя+локация из инвентаря. Несколько IP → альбом. Копия уходит в Drive |
| `/diag <ip\|имя>` | Полная диагностика: ping, MAC, открытые порты (иконки 📹554/🌐80/…), ONVIF-инфо + строка инвентаря «имя · локация · свитч/порт». Вывод моно-таблицей |
| `/info <ip\|имя>` | ONVIF: модель/прошивка/серийник. Мгновенно из кэша, кнопка «🔄 Обновить (живой ONVIF)» |
| `/ping <ip>` | Быстрая проверка доступности |
| `/rtsp <ip\|имя>` | Готовая RTSP-ссылка для VLC + кнопка на веб-интерфейс камеры |
| `/find [подсеть]` | Скан подсети: живой прогресс, страницы ◀▶, кнопки подсетей, ⏹ Стоп, ETA. Помечает «известная/НОВАЯ» по инвентарю. Только разрешённые подсети |
| `/lapse <ip> [n]` | Серия из n кадров альбомом — видно, живой ли поток |
| `/compare <ip1> <ip2>` | Два снимка рядом для сверки ракурсов |
| `/clip <ip> [сек]` | Видеоклип с RTSP (требуется ffmpeg: `winget install ffmpeg`) |
